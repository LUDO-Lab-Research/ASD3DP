# Run: pretrained encoder kNN

This package contains the audited frozen-encoder kNN and residual-view code,
compact score evidence, and external model locks. Generated stereo staging,
embedding caches, upstream repositories, and pretrained weights are excluded.

The historical rear execution source is not completely byte-identical to every
current portable file. That boundary is retained in
[`ERRATA_AND_PROVENANCE.md`](../ERRATA_AND_PROVENANCE.md).
